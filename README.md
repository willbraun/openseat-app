Overview

OpenSeat is an app designed to help you meet people in your area that share the same interests as you. You can create events as well as sign up for events that others have posted. 

Features
- Mobile responsive application
- View basic event information and search while logged out
- Create an account to create and sign up for events
- Find events on the Discover page by selecting a location, radius, and search phrase
- View participants on the event
- Fill a seat on the event
- Cancel a seat on the event
- Text notifications to creators when someone else has filled or canceled a seat
- Create an event
- Edit an existing event
- Delete an event
- View all events you are going to on My Seats
- View all events you have created on My Events

Technologies used
React, Python, Django, Django REST Framework, Google APIs (Distance Matrix, Maps, Places, Geocoding), Twilio API, React Router, React Bootstrap, React Google Places Autocomplete, Fuse.js, Compressorjs, JS-cookie, Date-fns
